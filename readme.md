# 💱 Conversor de Moedas

![Preview do Projeto](./img/Captura%20de%20tela%202026-03-15%20135240.png)

Projeto de **Conversão de Moedas** desenvolvido durante minha jornada de estudos com a Rocketseat, utilizando **JavaScript, HTML e CSS**.

O objetivo do projeto foi praticar conceitos fundamentais do desenvolvimento web, aplicando lógica de programação e interação com o usuário para criar uma aplicação funcional no navegador.

---

## 🚀 Funcionalidades

- Conversão de valores entre moedas
- Interface simples e intuitiva
- Interação dinâmica com JavaScript
- Atualização de valores na tela

---

## 🛠️ Tecnologias utilizadas

- **HTML5** → Estrutura da aplicação  
- **CSS3** → Estilização da interface  
- **JavaScript (ES6+)** → Lógica da aplicação e manipulação do DOM
